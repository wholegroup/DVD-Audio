How to install
  
 
Use the following steps to install Audio SixPack.
  • If you've downloaded the self - extracting installation package named AudioSixPackXXXX.exe just execute it and follow the setup instructions;
  • If you've downloaded the compressed archive file named like "AudioSixPackXXXX.zip" unpack it to the temporary folder and execute the AudioSixPackXXXX.exe file;
  • After the successful installation run the program from new "Audio SixPack" program group.
