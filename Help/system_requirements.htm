System Requirements
  
 
The listing below specifies the minimum system requirements for running Audio SixPack:

Software Requirements
  • Microsoft Windows 2000/XP or later
Hardware Requirements
  • Intel Pentium or compatible processor
  • 128 MB of RAM
  • 10 GB free disk space
  • One or more DVD recorders
